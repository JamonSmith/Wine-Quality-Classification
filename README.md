# Wine-Quality-Classification
This Jupyter Notebook Project explores a dataset of red wines using data mining and machine learning techniques. This project includes Exploratory Data Analysis, Regression Analysis, and Clustering (K-Means and Agglomerative Hierarchical).

Task 1:
  Exploratory Data Analysis - 
  1a. Description and distribution of variables (e.g. number of input variables and output variables)
      using data set summary and histograms of each variable.

  1b. Checking for data quality issues (e.g. duplicates and null entries).

  1c. Finding noticeable relationships between variables using a heatmap visual.

  1d. Coming up with my own questions that came from this initial inspection of the data and creating a 
      couple of visuals to go with them.

Task 2:
  Regression Analysis -
  2a. Perform multiple linear regression with pH as the output variable and the rest as input variables.
      Show results of this regression (e.g. coefficient, error, t-values for each input variable).
      Describe findings.

  2b. Create diagnostic plots of the linear regression. Describe any thing that looks problematic. 
      Look for any outliers.

  2c. Create three more linear regressions with alcohol as the output variable and some other variables
      inputs (e.g. 'density sugar' as a combination of 'density' and 'residual sugar'). Describe 
      findings.

Task 3:
  Clustering -
  3a. Create a K-Means clustering model with randomly placed centroids and iterate until centroids 
      no longer change. Show the error for each k used in a line graph and determine the elbow point 
      in the visual.

  3b. Create two agglomerative hierarchical clustering models (single link and complete link) and 
      calculate each of their silhouette scores. compare the two models to determine which model is
      better between the two.
